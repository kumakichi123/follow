# 追客プロの初期セットアップ

15分で追客プロを稼働させるためのチェックリストです。この手順を完了すれば、すぐにWeb見積もりを発行できます。

<StepCard title="Step 1" summary="アカウントを有効化">
招待メールのリンクを開き、二段階認証を設定します（認証アプリ推奨）。権限ロールは「Owner」であることを確認してください。
</StepCard>

<StepCard title="Step 2" summary="顧客テンプレを読み込む">
settings/data-import にあるCSVアップローダーで顧客マスタをインポートします。最初の100件で検証→全件アップロードの流れがおすすめです。
</StepCard>

<StepCard title="Step 3" summary="松竹梅テンプレを同期">
テンプレート管理から「松・竹・梅」を有効化し、単価表をアップロードします。ここで登録した金額がWeb見積もりに反映されます。
</StepCard>

---

これで初期設定は完了です。トライアル期間中はサポートチャットで画面共有も可能なので、不明点があれば遠慮なく連絡してください。
